# Универсальная платформа Телематики — техническое задание и артефакты проектирования
`Заказчик: «ID Expert» (Априорные решения машин). Автор: Бужор Александра`.

Проектная документация и сопутствующие артефакты для разработки универсальной платформы
**Телематики** — системы сбора, хранения, обработки и предоставления данных о работе
разнородного парка техники: тракторов (ДВС), электрических вилочных погрузчиков, тележек
и роботов (ВАТС).

Ключевая идея — **универсальность**: единая модель данных и единый API одинаково работают
с любым типом техники независимо от производителя, типа привода и степени автоматизации.
Добавление нового типа ТС или нового параметра телеметрии не требует изменения структуры БД
и переписывания API.

## Основные материалы

**`docs/TZ_Telematics.docx`** — техническое задание и функциональная спецификация (19 страниц).
Содержит: роли пользователей и бизнес-требования; полный перечень телеметрических параметров
(общие / ДВС / EV / ВАТС / специфичные) и каталог событий с границами значений; пользовательские
сценарии (User Stories) с приоритизацией по MoSCoW и критериями Given/When/Then; функциональные
и нефункциональные требования; архитектуру сбора и хранения; универсальную модель данных
с ER-диаграммой; спецификацию API и форматы входных/выходных данных; прототипы интерфейсов;
требования к инфраструктуре и план поэтапной реализации.

**`api/openapi.yaml`** — спецификация REST API (OpenAPI 3.0.3): 17 эндпоинтов, 21 схема.
Покрывает приём телеметрии и событий от устройств (ingestion), реестр ТС, выдачу временных рядов
метрик, состояние парка для карты, события и нарушения, миссии ВАТС, геозоны, правила алертов,
отчёты и справочники. Аутентификация: JWT (пользователи) и device-token / mTLS (устройства).

**`db/schema.sql`** — DDL схемы данных для PostgreSQL / TimescaleDB: 19 таблиц. Универсальная
расширяемая модель (справочники типов ТС/датчиков/событий, JSONB для вариативных атрибутов),
партиционирование метрик и событий по времени, BRIN и составные индексы, снимок последнего
состояния ТС с PostGIS-геометрией, миссии, геозоны, правила алертов, роли/пользователи. Включены
сид-данные справочников.
